Pinterest Scraper

localhost:8080

How to use

Search

Paste a Pinterest pin URL or type a search query (e.g. "pottery", "street fashion"). Set the number of images to find, then click Scrape.

Explore

Click any image to drill into that pin and find similar images. Use the back arrow to return to previous results.

Select

Click the Select button to enter select mode, or hold Shift temporarily. Click images to select them. Use Select All to grab everything visible.

Download

Type a collection name (e.g. "pottery") and click Download. Images are saved to ./downloads/<collection>/ next to server.py. Previously downloaded pins show a green border.
